# Retail Sales Analysis

## Project Overview

This project analyzes retail sales transactions to identify revenue trends, customer purchasing behavior, product performance, and country-wise sales distribution.

## Tools Used

* Excel (Data Cleaning)
* MySQL (Data Storage & SQL Analysis)
* Power BI (Interactive Dashboard)
* Python (EDA & Visualization)
* Git & GitHub

## Dataset

Online Retail Dataset containing transactional sales data including Invoice Number, Product Details, Quantity, Customer ID, Country, Revenue, Month, and Day.

## Key KPIs

* Total Revenue: 9.75M
* Total Orders: 26K
* Total Customers: 4K
* Total Products: 4K

## Dashboard Insights

* Revenue trend by month
* Orders distribution by day
* Revenue by country
* Top-selling products
* Customer purchasing behavior
